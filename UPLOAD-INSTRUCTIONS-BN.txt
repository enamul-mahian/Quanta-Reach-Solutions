QUANTA REACH SOLUTIONS — DIRECT UPLOAD
======================================

1. এই ZIP-এর সব file সরাসরি hosting-এর public_html folder-এ Extract করুন।
2. নিশ্চিত করুন public_html/index.html, public_html/.htaccess,
   public_html/assets, public_html/src এবং public_html/runtime-config.js আছে।
3. ZIP-এর file যেন public_html/dist বা অন্য nested folder-এর ভিতরে না থাকে।
4. Cache/CDN purge করে website reload করুন।
5. Admin/CMS/Firestore চালাতে runtime-config.js-এ নিজের Firebase Web App values দিন।
6. Secret key, Firebase Admin credential, service-account JSON বা Cloudinary API
   Secret কখনো runtime-config.js-এ দেবেন না।

নোট: এই direct-upload release pinned frontend packages esm.sh CDN থেকে এবং
React Quill stylesheet jsDelivr থেকে load করে। Public website Firebase ছাড়াও
চলবে; Firebase না থাকলে Contact/Quote form email compose fallback ব্যবহার করবে।
